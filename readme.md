# jekyll-filename

Adds a filename attribute to `post` objects in Jekyll. 


## Installation

Copy `filename.rb` to `/_plugins/` in your Jekyll site.


## Usage

`{{post.filename}}` outputs something like `2013-05-12-my-birthday.md`
